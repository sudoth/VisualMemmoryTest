# VisualMemoryTest

Небольшая игра на Pygame для тренировки визуальной памяти. На экране отображаются семь шестиугольников (соты). Игра показывает последовательность подсветок, после чего игрок должен повторить ее кликами мыши.

## Особенности
- Классический режим "Simon": последовательность растет после каждого верного раунда.
- Подсветка сот при показе последовательности и при клике.
- Простая настройка цветов, размеров и задержек через `config.py`.

## Требования
- Python 3.8+
- pygame

Установка зависимостей:
```bash
uv sync
```

## Запуск
```bash
uv run run.py
```

## Управление
- ЛКМ по соте, чтобы повторить последовательность.
- Закрыть окно — выйти из игры.

## Настройки
Файл `config.py` содержит основные параметры:
- размеры окна (`WIDTH`, `HEIGHT`);
- цвета (`ORANGE`, `YELLOW`, `CLICK_HIGHLIGHT`, `BACKGROUND`, `EDGE`);
- геометрия сот (`DISTANCE`, `EDGE`);
- задержки (`DELAY`, `CLICK_DELAY`).

## Структура проекта
- `Honeycomb.py` — логика игры и отрисовка сот.
- `geometry.py` — геометрия шестиугольника и проверка попадания клика.
- `config.py` — конфигурация игры.
- `run.py` — точка входа.
- `snd1.wav`, `snd3.wav` — звуки успеха/ошибки.
